# Online Clothing Store

## Features
- User Registration & Login
- Product Search
- Category Filter
- Shopping Cart
- Checkout
- Order History
- Admin Dashboard
- Product CRUD
- Order Management

## Technologies
- PHP
- MySQL
- Bootstrap 5
- HTML
- CSS
- JavaScript

## Installation
1. Copy project to htdocs
2. Import shopping.sql
3. Start Apache & MySQL
4. Open http://localhost/cloth-store
